About Me

简介

现任职位：初灵信息 / 产品中心 / 高级UI设计师

现居杭州，08年本科毕业，至今已有11年设计工作经验，期间负责过设计团队的统筹工作，参与过大大小小几十个项目。

  • 目前负责公司主要项目的视觉效果呈现、交互动效设计、UI 规范的制定、打造产品品牌形象等工作，不定期在团队中进行培训和分享，致力于让设计更加规范、高效。
  • 曾是中国电信浙江分公司多个合作项目的指定设计师，服务过中国电信、中国建设银行、一汽大众、方太集团、华润燃气、迪卡侬、中国移动设计院等大企业客户。
  • 除了全职 UI 设计师职位，还在大学担任外聘教师一职，负责 UI 设计以及前端课程的编排和教授，课后学生反馈良好。
