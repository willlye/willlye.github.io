微加卡通角色设计

3D Design

项目描述

为提高微加品牌辨识度设计的 IP 形象。

在如何提升公司产品的品牌辨识度问题上，我有过一些思考。通过对大部分竞品的分析，我发现目前各家的设计都是在朝着界面简洁、色彩明快的方向走，这就让大家的产品同质化问题更加严重。

现在 C4D 在 UI 设计上被使用得越来越多，但是目前竞品当中还没有发现有哪家在使用 IP 形象的，所以我决定开始尝试为微加打造一对职场白领的卡通角色，使之逐渐成为能够代表微加产品的 IP 形象。

目前两个角色在 Banner 及移动端一些页面上已经开始使用，希望通过不断地增加曝光率使用户今后在看到这两个角色的时候，就立刻能联想到微加。
