微加官网改版

Web Design

项目背景

微加项目自成立以来，网站经历过大大小小数次改版，经过数位设计师之手，由于前期没有形成规范，这就导致整个网站看起来非常凌乱，没有统一的风格和交互逻辑，甚至许多页面的字号都存在偏差。

为了避免后期再出现风格混乱的问题，首先由我在界面简洁、交互清晰的基本原则基础上，将“更轻、更快、更优雅”作为目标，独自进行网站改版的设计。在设计的同时也开始制定整个网站的规范，用 Sketch 将色彩、字号、圆角、间距等元素提前定义好，做成组件来进行管理。

我希望用户在浏览网站时不会感觉沉闷和繁琐，所以在简化了页面层级和交互逻辑的基础上，我还要求加入一些微动效，尽量使用户在操作页面时能够感觉轻量、流畅，甚至赏心悦目。
